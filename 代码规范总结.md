初始字典变量定义总结
--------
初始化字典变量时，不需要重复定义变量名

**例子分析**

```php
$email_content['identify'] = $this->getIdentifyCode();
$email_content['code'] = $this->getIdentifyCode();
```
不需要重复定义变量名，统一定义

```php
$email_content[
    'identify'=> $this->getIdentifyCode(),
    'code' => $this->getIdentifyCode()
]
```
